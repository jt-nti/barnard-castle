# barnard-castle

Hack to scan NHS COVID-19 check-in QR codes

Built with:

- [Bulma](https://bulma.io/)
- [Font Awesome](https://fontawesome.com/)
- [jsQR](https://github.com/cozmo/jsQR)
- [jwt-decode](https://github.com/auth0/jwt-decode)

## Development

To test locally:

```
python3 -m http.server
```
